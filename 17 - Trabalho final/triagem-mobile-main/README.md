# Triagem Mobile - FalAI Doutor

Demo Godot 4.x em 16:9 para PC e mobile. O fluxo simula um segundo frontend para o FalAI Doutor:

- Paciente caminha até o totem e envia sintomas.
- Profissional de saúde caminha até o computador e analisa a triagem.
- O jogo envia `x-application-key` nas chamadas HTTP ao backend NestJS.

## Controles

- PC: WASD ou setas para mover, `E` para interagir, `Tab` para alternar jogador.
- Mobile: botoes de direcao, `Interagir` e `Trocar` na tela.
